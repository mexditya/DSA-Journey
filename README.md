This repository contains my solutions, notes, and logic explanations while following **Rohit Negi's DSA Playlist**.

I started this repository to keep track of my progress, improve consistency, and create a revision resource that I can come back to anytime. Each lecture is organized separately and contains the questions solved during that lecture along with the logic used to solve them.
