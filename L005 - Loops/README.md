# L005 - Loops

This folder contains basic problems based on loops in C++. These programs helped me understand how to use `while` and `for` loops for repetition, calculations, and number-based problems.

## Questions Solved

### 1. Factorial of a Number

Calculates the factorial of a given number.

**Logic:**

* Initialize the factorial as 1.
* Multiply it by every number from 1 to n.
* Print the final result.

**What I learned:**

* Using loops for repetitive calculations.
* Accumulating results inside a loop.

**File:** `Factorial.cpp`

---
